# Installing Zen Notes via Sine

The easiest way to install Zen Notes is through the **Sine mod manager**!

## Prerequisites

1. **Install Sine** first by following the instructions at: https://github.com/CosmoCreeper/Sine

## Installation via Sine Marketplace

### Option 1: From Marketplace (Once Published)

1. Open Zen Browser
2. Go to `Settings` → `Sine`
3. Search for "Zen Notes" in the marketplace
4. Click `Install`
5. Restart Zen Browser
6. Press `Ctrl+Shift+N` to open the notes panel!

### Option 2: Install from GitHub (Before Marketplace Publication)

1. Open Zen Browser
2. Go to `Settings` → `Sine`
3. Click on "Add Unpublished Mod"
4. Enter the repository name: `yourusername/zen-notes-mod`
5. Click `Add`
6. Restart Zen Browser
7. Press `Ctrl+Shift+N` to open the notes panel!

## Publishing to Sine Marketplace

To publish this mod to the official Sine marketplace:

1. **Fork this repository** to your GitHub account
2. **Add a preview image**: Create a preview.png (recommended size: 1280x720px) and add it to the repository
3. **Update theme.json**: Replace `yourusername` with your actual GitHub username in:
   - `homepage`
   - `updateURL`
   - `image`
   - `readme`

4. **Submit to Sine**: Create a new issue at https://github.com/CosmoCreeper/Sine/issues/new?template=add-theme.yml with:
   - Mod name: Zen Notes
   - Repository: `yourusername/zen-notes-mod`
   - Description: Copy from theme.json
   - Screenshot: Link to your preview.png

5. **Wait for approval**: The Sine maintainers typically respond within 24 hours!

## Auto-Updates

Once installed via Sine:
- **Automatic updates**: Sine pulls updates directly from your GitHub repository
- **No update requests needed**: Users get the latest version automatically
- **Version control**: Managed through your repository releases

## Troubleshooting

### Notes panel doesn't appear
1. Make sure Sine is properly installed
2. Check `Settings` → `Sine` to verify Zen Notes is enabled
3. Restart Zen Browser completely
4. Press `Ctrl+Shift+N` (or `Cmd+Shift+N` on Mac)

### Can't find in marketplace
- The mod hasn't been published yet
- Use "Add Unpublished Mod" method (Option 2 above)

### Updates not working
- Check your internet connection
- Go to `Settings` → `Sine` → Check for updates
- Make sure auto-update is enabled for Zen Notes
