# Changelog

All notable changes to the Zen Notes mod will be documented in this file.

## [1.0.0] - 2025-01-01

### Initial Release

#### Features
- ✅ Sidebar-based notes panel with toggle functionality
- ✅ Create, edit, and delete notes
- ✅ Auto-save with 2-second debounce
- ✅ Link notes to specific URLs (context awareness)
- ✅ Scratchpad mode for general notes
- ✅ Tag system for organization
- ✅ Full-text search across notes
- ✅ Three filter modes: All, Current Page, Scratchpad
- ✅ Keyboard shortcut: Ctrl+Shift+N
- ✅ Local storage (privacy-first)
- ✅ Dark/light theme support
- ✅ Responsive design
- ✅ Character counter
- ✅ Smooth animations
- ✅ Accessible keyboard navigation

#### Technical
- Storage using localStorage
- Maximum 1000 notes
- Each note includes: ID, content, URL, tags, created date, modified date
- Performance optimized with debounced auto-save
- Minimal memory footprint when inactive

#### Documentation
- Complete README with installation instructions
- Quick Start guide
- Example configuration files
- Troubleshooting section

#### Known Limitations
- No export/import functionality yet
- No rich text formatting
- No cloud sync
- No note folders/categories
- Single user (no collaboration)
